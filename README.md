## Xworm RAT 的主要功能 远程控制

- 允许攻击者执行命令、窃取文件和监视活动。

- 键盘记录和凭据窃取

- 记录按键以窃取密码、信用卡详细信息和其他敏感数据。

- 屏幕截图和网络摄像头访问

- 未经同意截取屏幕截图并激活受害者的网络摄像头。

- 僵尸网络和 DDoS 功能

- 可以将受感染的设备变成机器人，以发起分布式拒绝服务 (DDoS) 攻击。

- 持久性机制

- 隐藏在系统中（通过注册表编辑、计划任务）以在重启后存活。

- 通过漏洞和 USB 驱动器传播

- 可以通过网络钓鱼电子邮件、恶意下载或 USB 设备传播。

- 加密货币矿工

- 一些变体使用受害者的硬件秘密挖掘加密货币（Monero、比特币）。
